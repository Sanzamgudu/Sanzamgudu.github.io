Frontline support (Level 1)

- Responsible for handling basic technical issues such as common software glitches or minor user errors.

- Responsible for providing solutions to errors that require predefined knowledge base.

- Taking down of notes to close the ticket

- Should the situation go otherwise then the issue will be escalated to the second level.

 

Second Level support (Level 2)

-        This level is responsible for handling more complex technical issues than level 1, unresolved issues.

-        Tries to implement advanced troubleshooting skills and solutions.

-        If the issue is resolved, close the ticket with detailed resolution steps.

-        Should the issue be unresolved then it is passed on to the 3rd level support.

Third level Support (Level 3)

-        Address highly technical and critical issues that require expert intervention.

-        Analyse the issue thoroughly using advanced troubleshooting skills and solutions.

-        When the issue is resolved, note down the solution and close the ticket.

-        If the issue is unresolved, then it is passed to the development team, for debugging which would be the development team.
